# 💬 Affirmation App

A simple Android app built using **Jetpack Compose** that displays a list of motivational affirmations with beautiful images.

This project is part of my learning journey through the [Android Basics with Compose](https://developer.android.com/courses/android-basics-compose/course) course by Google.

---

## 📱 Features

- Scrollable list of affirmations using `LazyColumn`
- Display text and images in a card layout
- Modern UI built with **Jetpack Compose**
- Responsive padding using `WindowInsets` to respect system bars
